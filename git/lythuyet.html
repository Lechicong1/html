Git/github

Git clone link : sao chép src code từ github về folder

Git init : khởi tạo kho lưu trữ git
Git add . : thêm src code vào git
Git commit -m: mô tả cho lần sửa đổi
git branch -M main : chuyển sang nhánh main
git remote add link : cho biết git sẽ đẩy code đến link nào
git push : đẩy code lên github git pull: lấy code từ github về
